# 思维导图独立全屏预览窗

日期：2026-07-17  
范围：PyQt 主应用右侧「思维导图」Tab  
风格：沿用 Corporate Clean

## 目标

右侧窄栏无法舒适阅读 Mermaid 导图。提供独立全屏/最大化预览窗，不打断视频播放。

## 方案

- 工具栏增加「全屏预览」按钮（无导图内容时提示先生成）
- 独立 `QDialog`（可最大化），主体为 `QWebEngineView`，复用 `mindmap_to_html`
- Esc / 关闭按钮退出；关闭后右侧 Tab 内容不变
- 预览窗内保留缩放（Ctrl+滚轮 + 工具栏），并支持拖拽平移画布
- 无 WebEngine 时预览窗降级为只读 Mermaid 源码
- 默认右侧面板宽度略增（420→480），主路径仍是全屏

## 非目标

- 不改 Electron 前端
- 不更换全局主题
- 生成完成后不自动弹全屏
