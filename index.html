Capabilities

核心能力

重活在本地 Python 管线完成；UI 用 PyQt6，可选 Electron 连接同一 FastAPI。

流式同声传译

yt-dlp 直链 → mpv 播放（原声可静音）→ faster-whisper ASR → 百炼等大模型翻译 → CosyVoice TTS，字幕实时叠加。

AI 一键总结

播放中累积双语字幕，调用大模型生成结构化 Markdown；可尝试拉取 B 站已有字幕作补充。

思维导图

基于字幕与总结生成 Mermaid mindmap，QWebEngine 预览，支持导出 Markdown / HTML。

多厂商大模型

可视化配置百炼 / OpenAI / DeepSeek / 自定义 OpenAI 兼容；翻译、总结、导图模型可分别指定。

Stack

技术亮点

Python 负责播放、ASR、翻译与 TTS；可选 Electron 前端共用 API。

PyQt6· mpv· faster-whisper· yt-dlp· FastAPI· DashScope / OpenAI 兼容· CosyVoice· Electron（可选）

Get started

安装与使用

需要 Python 3、mpv、ffmpeg，以及至少一个大模型 API Key（见 .env.example）。

01 · 依赖
pip install -r requirements.txt
brew install mpv ffmpeg
cp .env.example .env   # 填入 LLM_API_KEY 等
02 · PyQt6 完整应用（推荐）
python main.py
03 · Electron + API（可选）
# 终端 1
python run_api.py
# 终端 2
cd electron && npm install && npm run electron

视频播放在 PyQt6 主应用中；Electron 侧重 AI 配置、总结与思维导图。完整说明见 README。
